PROJECT DETAILS

Autonomous Parking System is a system to inquire the details of a car when parked in a parking lot. It provides the details like Slot Number, Vehicle Plate Number, Date, Time of Parking and Image Proof. It dynamically stores the data to keep the record, prevent data loss, usage in urgent times.



TOOLS USED

We have used OpenCV in retriving the licence plate either by capturing through camera or using a image. Then importing several libraries like cv2, imutils, pytesseract, datetime, to perform a series of convenient operations to make basic image processing functions such as translation, rotation, resizing, sorting contours, detecting edges, cropping image etc.

The entire path from getting the driver plate and other useful details such as date and time of parking to storing the data to prevent data loss incase of program failure is done using Python. The running of the program repeatedly until notified to end is also done using Python.

It forms the skeleton of the website as well as give some styling of element.

Retrieving the data from the camera to the website, JavaScript creates the table dynamic, and updates it as well.

For saving the time and having a shorter and easy to read code, the Bootstrap framework is used.

Data is stored in the JSON document. This helps as the data remains stored even if the power cuts off, although it's not the best way out for this and soon be resolved.
